Minha Jornada no Desenvolvimento

Meu interesse pela área de desenvolvimento começou quando tive meu primeiro contato com programaçãodurante o curso técnico de Informática no ensino médio. Fiquei fascinado pela capacidade decriar e dar vida a um software. Após passar por alguns altos e baixos da vida, decidi seguir esse caminho. Deixei minha família e meu antigo emprego na bahia, e com o apoio dos meus tios, com quem moro atualmente, me mudei para São Paulo com o objetivo de realizar meu sonho de me tornarum desenvolvedor.

Ao chegar em São Paulo, comecei a trabalhar como garçom aos finais de semana, me matriculei no curso tecnólogo de Análise e Desenvolvimento de Sistemas e continuei meus estudos de programação. Desde então, venho desenvolvendo minhas habilidades e me apaixonando cada vez mais pela área. Gosto de comparar a programação a um jogo do estilo Souls-like — desafiador, porém recompensador. A cada obstáculo, evoluo, adquiro novas habilidades e, mesmo diante defalhas, a sensação de superação é incrível. Sinto que, a cada desafio vencido, me torno uma versão melhor de mim mesmo.

Por que me dar uma oportunidade?

Sou uma pessoa comunicativa, persistente e focada no que faço, pois acredito que todo esforço um dia será recompensado. Estou sempre buscando adquirir e aperfeiçoar meus conhecimentos, seja na faculdade, em cursos ou desenvolvendo projetos pessoais. Tenho uma grande vontade de aprender e evoluir, e por isso estou disposto até mesmo a aceitar estágios não remunerados. Embora eu possa não ser o programador mais experiente, garanto que serei o mais dedicado, e estou confiante de que não decepcionarei.

Tecnologias Dominadas

HTML (HyperText Markup Language) linguagem de marcação usada para criar e estruturar páginas na web

CSS (Cascading Style Sheets) linguagem que define o estilo e o layout das páginas web,

JS (JavaScript) linguagem de programação que torna páginas web interativas e dinâmicas.

Git sistema de controle de versão para gerenciar código. & GitHub plataforma para hospedar e colaborar em repositórios Git.

Tecnologias Sendo Dominadas

React é uma biblioteca JavaScript que facilita a criação de interfaces dinâmicas e reutilizáveis.

Tailwind CSS é um framework utilitário que agiliza o design com classes predefinidas no HTML.

TypeScript adiciona tipagem ao JavaScript, tornando o código mais seguro e organizado.
